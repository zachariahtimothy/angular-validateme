Overview

Angular-ValidateMe is a configurable directive for providing simple but extendable inline form validation for AngularJS applications.

Dependencies

Basically it does not depend on any libraries except AngularJS itself.
For validation theming to appear properly out of the box, you will need to include Twitter Bootstrap CSS.

Features

Allows you to utilize the built-in Angular validation with the goodness of Bootstrap validation styling.
I was frustrated by the idea of adding 4 spans for every element I wanted to validate, specially because many of them are identical. Now you can skip the manual addition of error handling if you are using the built-in Angular validations (like ng-minlength="2"). If you write a directive for custom validation, no problem! Simply add those custom spans and enjoy a better life.

Get started

  1. Include Angular.js in your project

    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.23/angular.min.js"></script>

    Optionally include Bootstrap CSS for theming

    <link href="//netdna.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css" rel="stylesheet">
  2. Install angular-validateme via bower or download latest zip 
    bower install angular-validateme
  3. Include angular-validateme into your project 
    <script src="bower_components/angular-validateme/dist/js/validateme.js"></script>
  4. Define angular app and add "validateme" to dependencies 
    <html ng-app="app">
    var app = angular.module("app", ["validateMe"]);
  5. Optionallly override styles or validation messaging in app.config.: 
    app.config(function(validationOptionsProvider) {
  validationOptionsProvider.containerClasses.success = 'yep-success';
  validationOptionsProvider.validationElement.setMessage('required', 'FOOL! You must fill this out or pay the price!');
});
  6. Markup element that should be editable 
    <div ng-controller="Ctrl">
  <input type="text" name="myfield" ng-minlength="5" validateme>
</div>
  7. Sleep well knowing your entries are valid!
